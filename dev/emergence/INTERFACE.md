# INTERFACE.md

## Project Emergence

### Version 0.1

> The interface is not where writing happens.
>
> It is where thought becomes observable.

---

# Purpose

The interface exists to reveal the genealogy of thought while remaining quieter than the thought itself.

Every interaction should preserve flow.

Every visual element should justify its existence.

The interface never asks the user to think about the interface.

---

# Primary Principle

The writer should always know:

* Where this thought came from.
* Where it currently exists.
* What may grow from it.

Nothing else is essential.

---

# Interface Philosophy

Most writing applications organize files.

Project Emergence organizes living thought.

Therefore, navigation follows ancestry rather than folders.

The interface is not document-centric.

It is lineage-centric.

---

# Primary Workspace

The interface revolves around a single object.

The current thought.

Nothing competes for attention.

No permanent sidebars full of tools.

No inspector panels by default.

No dashboard overload.

The writer enters directly into thinking.

---

# Interface Regions

## 1. Thought

The current node.

This occupies the visual center.

It is always the dominant element.

---

## 2. Context

A quiet indication of ancestry.

Not the entire genealogy.

Only enough to answer:

"Where am I?"

Example:

🌱 Emergence

↓

🌿 Reflection

↓

(Current)

---

## 3. Actions

Actions appear only when relevant.

The interface prefers absence over disabled controls.

Possible actions include:

Seal

Create Descendant

Publish

Reflect

Actions should feel like natural continuations of thought rather than software commands.

---

## 4. Genealogy

The genealogy is contextual.

It is never the default focus.

The writer enters to write.

Not to browse history.

The genealogy appears only when intentionally explored.

---

# Writing Experience

The editor should resemble paper more than software.

Formatting is secondary.

Expression is primary.

The cursor should feel alone.

The page should breathe.

---

# Creating an Emergence

The first experience defines the philosophy.

The interface opens into emptiness.

No project creation.

No folder selection.

No metadata.

Only silence.

The first written word creates an editable Expression.

Only Seal creates an Emergence.

Nothing exists before the thought.

---

# Seal

Seal is the most important interaction.

It should feel ceremonial without becoming theatrical.

After sealing

the Expression becomes an Emergence.

Editing stops because the editable object no longer exists.

The thought becomes history.

The interface communicates permanence through restraint.

No confirmation dialog.

No celebration.

Only recognition.

---

# Descendant

A Descendant never replaces its parent.

Creating one should feel like continuing a conversation.

The parent remains visible.

The descendant opens beside its ancestry conceptually, not visually.

The writer never experiences deletion.

Only continuation.

---

# Reflection

Reflection is reading with intention.

Reflection mode reduces editing affordances.

The interface privileges observation over production.

---

# Publication

Publication changes visibility.

Nothing else.

A published thought continues belonging to its genealogy.

Readers see one projection.

The lineage remains intact.

---

# Navigation

Navigation follows thought.

Possible navigational directions are:

Ancestor

Descendant

Sibling

Lineage

Publication

Chronological navigation is secondary.

---

# Progressive Disclosure

The interface never reveals maximum complexity immediately.

Depth appears only when curiosity requests it.

This principle applies everywhere.

---

# Information Density

The interface defaults to low density.

Every additional visual element must justify its cognitive cost.

Silence is the baseline.

Information is the exception.

---

# Controls

Controls should almost disappear.

Icons support recognition.

Words carry meaning.

Buttons should communicate intention rather than software functionality.

Examples:

Seal

Descend

Reflect

Publish

Not:

Save

Edit

Commit

Submit

---

# Notifications

Notifications should be rare.

The system should avoid interrupting thought.

Whenever possible:

communicate through state

rather than alerts.

---

# Search

Search should prioritize concepts over filenames.

The user remembers ideas.

Not where they stored them.

Searching for:

"identity"

should reveal lineages,

not merely text matches.

---

# Genealogy View

The genealogy is an exploratory environment.

Not a dashboard.

It should encourage curiosity.

Never obligation.

The complete tree exists.

The interface reveals only meaningful portions.

---

# Reading

Reading and writing are equally important.

The interface should support contemplation.

Not merely production.

Reading one's own ancestry should feel like meeting previous versions of oneself.

---

# Mobile

Mobile is not a reduced desktop.

Mobile is the natural habitat of Emergence.

Emergences happen while walking,

traveling,

waiting,

living.

The capture experience should therefore be simpler than desktop.

Desktop favors reflection.

Mobile favors emergence.

---

# Desktop

Desktop privileges synthesis.

Comparisons.

Reflection.

Publishing.

Large genealogies.

Multiple descendants.

Desktop is where understanding matures.

---

# Accessibility

Meaning must never depend exclusively on:

Color.

Motion.

Icons.

Every state must remain understandable through structure alone.

---

# Failure

Nothing meaningful should ever disappear.

Undo exists.

History exists.

Genealogy exists.

Loss should be practically impossible.

The writer should develop complete trust in the system.

---

# Interface Test

Every interaction should satisfy the following questions:

Does this reduce cognitive friction?

Does this preserve ancestry?

Does this encourage exploration?

Does this remain quieter than the thought?

Would removing it make the philosophy weaker?

If not,

it should not exist.

---

# Final Principle

The interface should disappear.

The genealogy should remain.

The thought should endure.
